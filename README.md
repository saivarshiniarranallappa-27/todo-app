# 📝 To-Do Web Application

## 📌 Description
This is a simple To-Do web application built using Python (Flask) and SQLite.  
It allows users to add, view, edit, and delete tasks.

---

## 🚀 Features
- Add new tasks
- View all tasks
- Edit existing tasks
- Delete tasks
- Data stored using SQLite database

---

## 🛠️ Tech Stack
- Python (Flask)
- HTML
- CSS
- SQLite

---

## ▶️ How to Run

1. Clone the repository:
```
git clone https://github.com/your-username/todo-app.git
```

2. Go to project folder:
```
cd todo-app
```

3. Run the application:
```
python app.py
```

4. Open in browser:
```
http://127.0.0.1:5000
```
